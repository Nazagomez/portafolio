Hogar de libros System

Hogar de Libros is an informative and administrative web system developed for the Nicoya Library. It simplifies the management of book loans, returns, and user reservations, while also serving as a platform to share important announcements, cultural events, and educational programs. Users can explore available books, manage their current loans, and stay informed about library activities through a modern and intuitive interface. The platform includes role-based access, allowing administrators to oversee and manage inventory and user activity, while general users have access to personalized features. The system enforces library policies, such as restrictions on overdue loans or reservation limits, to maintain order and fairness.

Tools & Technologies

  • React vite + Flowbite
  • Node.js + Nest.js
  • MySQL Database
  • TypeScript
  • Responsive Design with Dark Mode
